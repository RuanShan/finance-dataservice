
YahooFinance Module: http://www.transparentech.com/opensource/yahoofinance

gem install yahoofinance

Ticker
---------
id integer - autogenerated
name string - e.g. AKAM, AKAM120317C00021000, EUR/AUD
security_type string - e.g. Stock, Option, Currency
underlying string - e.g. AKAM, AKAM120317C00021000, EUR/AUD

Indicator
---------
id :integer - autogenerated
indicator_type :string - e.g. open, close, high, low, ticker, volume, adj_close // atr, ma200
date :date - 
value :integer - 

rails generate model Technical indicator_type:string date:date value:integer ticker:references
rails generate controller Technicals
